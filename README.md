# Cleveland Parent News

A local news aggregator and blog focused on Cleveland, Ohio happenings for parents.

## Project Goal

Collect, curate, and deliver relevant local news, events, and information specifically tailored for parents in the Cleveland area.

## Data Sources

- Local news RSS feeds
- Community calendars
- School district announcements
- Parent-focused event listings

## Tech Stack

- Python for data collection
- RSS feed aggregation
- Web scraping utilities
- Content curation pipeline

---

*Built with OpenCode + GLM-5*
